# FIRE ALARM SYSTEM
A DLD project ie Logic Based Fire Alarm System Equiped with buzzers

## Components
1. 10K THERMISTOR
2. LM358 OPERATIONAL AMPLIFIER
3. 4.7K RESISTOR
4. 10K POTENTIOMETER
5. 5V BUZZER
6. CONNECTING WIRES
7. BREADBOARD
8. 5V POWER SUPPLY
